# Snake Mania - EK Gaming Katha 🐍🎮

A fun and engaging browser-based Snake Game that brings back the retro vibes! Built using HTML, CSS, and JavaScript, this game is simple to play yet challenging to master. Navigate the snake, eat the food, and aim for the highest score without colliding with walls or yourself!

---

## Features 🚀
- Real-time gameplay with smooth snake movement.
- Score tracking and high score storage using `localStorage`.
- Retro-inspired visuals with modern UI elements.
- Background music and sound effects for an immersive experience.

---

## Technologies Used 🛠️
- **HTML**: Structure of the game.
- **CSS**: Styling for the game board and elements.
- **JavaScript**: Core game logic, including collision detection, movement, and score management.

---

## How to Play 🎯
1. Use the arrow keys (`↑`, `↓`, `←`, `→`) to control the snake.
2. Eat food to grow longer and increase your score.
3. Avoid hitting the walls or the snake's body.
4. Aim for the highest score and beat your previous record!
